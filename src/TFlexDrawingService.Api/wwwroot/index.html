Инженерный центр

Выберите раздел для работы с чертежами, расчетами и проектами

Раздел позволяет выбрать проект, шаблон, формат файла и параметры лифтовой шахты. После заполнения параметров пользователь может сохранить конфигурацию и сформировать готовый чертеж.

Перейти к созданию

Раздел предназначен для расчета базовой стоимости по параметрам проекта. Пользователь может указать исходные данные, получить расчет и использовать его при подготовке коммерческого предложения.

Открыть расчет

В личном кабинете пользователь может просматривать проекты, сохраненные конфигурации, готовые файлы, историю действий и доступные операции для инженера, продавца или администратора.

Открыть кабинет

Сервис помогает инженерам быстро формировать чертежи, рассчитывать параметры и управлять проектами в едином рабочем пространстве.
